리디자인 이유와 목표

Reason

에버랜드라는 즐겁고 밝은 이미지가 전해지지 않고 좋은 컨텐츠와 정보들이 많은데
정작 매인 페이지에는 그것들이 들어나지 않는것이 아쉽다는 생각이 들어 이번 리디자인을 진행하게 되었습니다.

일러스트의 각 요소들은 에버랜드를
상징하는 것들이지만 정작 전체적인
분위기는 어둡다

이용하면 좋을 것 같은 컨텐츠나 정보들은 메뉴 속에
찾기 힘들게 들어있어서 아깝다

에버랜드 앱이라는 느낌이 들지 않는다

Target

메인페이지에 기존 유용한 정보와 컨텐츠들을 적극적으로 노출시키고
기존 웹사이트와의 통일감을 형성하면서 더 밝고 화사하게 디자인 하였습니다.

필요한걸 바로 검색할 수 있도록
검색버튼을 메인페이지 상단에 노출

현재 진행중인 할인 이벤트나 행사,
시즌별 패스티벌등을 홍보할 수 있는 슬라이드 제작

유용한 컨텐츠들은 유지하되 디자인만 변경

배경의 흰색과 로고에서 따온 밝은 색으로
기존대비 전체적인 느낌을 화사하게 만듬
